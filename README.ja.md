# Record · Finance

[English](README.md) · [中文](README.zh.md) · **日本語**

プライベートでオフラインファーストなマルチ通貨・純資産トラッカー。
あなたのデータが端末から外に出ることはありません。

**公開サイト：** https://record.finance

---

## 機能

- **マルチ通貨** —— CNY・USD・JPY・EUR・SGD・HKD、為替レートを自動取得
- **資産 / 負債アカウント**（現金・預金・投資・不動産・暗号資産・その他のカテゴリ付き）
- **純資産または総資産** —— トグル一つでアプリ全体の集計基準を切り替え
- **目標トラッキング** —— 到達予想時期の推定とマイルストーン演出付き
- **チャート** —— 残高推移、前年同期比、アカウント別ドリルダウン、通貨 / カテゴリ構成
- **今月 / 年初来 / 入金 vs 市場変動** の分解
- **ローカルファイル同期** —— File System Access API を利用し、任意のクラウド同期フォルダ（iCloud Drive、Dropbox、OneDrive）を指定。変更時に自動同期
- **ローリング・ローカルスナップショット**（直近 20 件の変更）でいつでも元に戻せる安心設計
- **PIN ロック** —— セッション内アンロック対応。プライバシーぼかしは長押しで一時的に確認可能
- **PWA** —— ホーム画面に追加可能、オフラインで動作（Service Worker）
- **多言語対応** —— English / 中文 / 日本語
- **ライト / ダークテーマ**、会計年度の開始月を設定可能

## 技術

ビルド工程もフレームワークもない、純粋な静的サイト。

- 単一の `index.html` 内の HTML / CSS / バニラ JS
- [Chart.js](https://www.chartjs.org/)（jsDelivr CDN 経由）
- [`@fawazahmed0/currency-api`](https://github.com/fawazahmed0/exchange-api)（jsDelivr CDN 経由、無料・API キー不要）
- File System Access API + IndexedDB（同期ハンドルの永続化用）
- Service Worker（オフライン対応とアセットキャッシュ用）
- Web Crypto API（SHA-256、PIN のハッシュ化用）

## プライバシー

- **バックエンドなし。** あなたのデータを保存するサーバーはありません。
- **分析なし、トラッキングなし、Cookie なし。**
- **アカウント / サインアップ不要。** ページを開けばすぐ使えます。
- **オープンソース。** すべてのコードがこのリポジトリにあります——自由に監査できます。
- すべてのデータはブラウザの LocalStorage と IndexedDB に保存されます。
- 任意の同期機能は、あなたが選んだ場所に JSON ファイルを 1 つ書き込むだけです。
- 為替レート API に渡るのはリクエストした日付のみ——個人情報は一切含みません。
